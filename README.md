To use [DeepL](https://www.deepl.com/en/translator) translation:

1. Install the [OmegaT](https://omegat.org/) [deepl-connector-plugin](https://github.com/omegat-org/deepl-connector-plugin).
2. Obtain a [DeepL Free API key](https://www.deepl.com/en/your-account/keys) and enter it in the deepl-connector-plugin settings.
3. Use this script to automatically translate the OmegaT project files.
